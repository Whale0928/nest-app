## Description

[Nest](https://github.com/nestjs/nest) framework project

**CRUD** / **postgreSQL+ORM** / **Auth** / **Log** 등을 학습

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
